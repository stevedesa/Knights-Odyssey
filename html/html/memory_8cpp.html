Author

Description
This function accepts a 2d pointer array and assigns 2d memory to the array of n rows and columns, based on the number of rows and columns provided by the user. The output matrix is a square matrix
Parameters
[in,out]array- accepts 2d pointer array, to assign dynamic memory.
[in]n- number of rows and columns of memory to assign.
Example
int** array;
int n = 5; //number of rows and columns
allocarray(array, n);
//now array is a dynamic 2d array of n rows and columns;

◆ Option1()

void Option1 ( )
Author
Steve Nathan de Sa
Description
This function runs the algorithm for option 1 of the program. It uses switch cases to keep on calling the options of the program, for the user to edit, until the user decides to call the tour function which prints out the tour and exits the program. It gives users the option to change the board size, or change the starting point of the tour. It also makes sure that all proceedings are withing appropriate bounds and checks to prevent overstepping of bounds. After the user is satisfied with it's requirements, it can set the function to call the tour function, to print out the knights tour.
Example
Option1(); //Runs Algorithm for Option 1
